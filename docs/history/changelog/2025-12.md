# 2025년 12월 변경 이력

## 2025-12-14 (Week 18)

### Resend SMTP 이메일 시스템 구축 완료 ✅

Supabase 기본 이메일 서비스의 rate limit 문제를 해결하기 위해 Resend 커스텀 SMTP를 설정.

#### 1. Resend 도메인 설정
**도메인**: `sign.g-plat.com`

**DNS 레코드 (Vercel)**:
- DKIM: `resend._domainkey.sign` → TXT 레코드
- SPF (MX): `send.sign` → MX 레코드 (priority 10)
- SPF (TXT): `send.sign` → TXT 레코드 (`v=spf1 include:amazonses.com ~all`)

**상태**: 모든 레코드 Verified ✅

#### 2. Supabase SMTP 설정
**Supabase Management API를 통해 설정**:

```json
{
  "smtp_host": "smtp.resend.com",
  "smtp_port": "465",
  "smtp_user": "resend",
  "smtp_pass": "re_MYaq2QQq_...",
  "smtp_admin_email": "noreply@sign.g-plat.com",
  "smtp_sender_name": "G-Plat",
  "rate_limit_email_sent": 30
}
```

**결과**:
- ✅ 회원가입 이메일 인증 정상 작동
- ✅ 비밀번호 재설정 이메일 발송 가능
- ✅ Rate limit: 30회/시간 (기존 2회에서 대폭 증가)

#### 3. API 키 관리
- **제한된 키**: `re_SRdcM3wN_...` (이메일 발송만 가능, 도메인 조회 불가)
- **전체 권한 키**: `re_MYaq2QQq_...` (도메인 인증 후 생성, 모든 권한)

**주의**: API 키는 도메인 인증 후 생성해야 해당 도메인에 대한 권한이 부여됨.

### Admin 지도 미리보기 로딩 수정 ✅

명함 생성 페이지에서 "지도를 불러오는 중..." 화면이 무한히 표시되는 문제 해결.

#### 문제 원인
Kakao Maps SDK의 `autoload=false` 옵션 사용 시, `window.kakao.maps.load()` 콜백이 완료되기 전에는 API를 사용할 수 없음.
기존 코드는 `window.kakao.maps` 존재 여부만 확인하여 SDK가 준비되지 않은 상태에서 지도 생성 시도.

#### 해결 방법

**파일**: `admin-app/index.html`
```html
<script>
  window.kakaoMapsReady = false;
  (function() {
    const script = document.createElement('script');
    script.src = '//dapi.kakao.com/v2/maps/sdk.js?appkey=...&autoload=false';
    script.onload = function() {
      window.kakao.maps.load(function() {
        window.kakaoMapsReady = true;
        window.dispatchEvent(new Event('kakao-maps-loaded'));
      });
    };
    document.head.appendChild(script);
  })();
</script>
```

**파일**: `admin-app/src/components/common/MapPreview.tsx`
```typescript
const [isMapReady, setIsMapReady] = useState(false)

useEffect(() => {
  const initMap = () => {
    if (!window.kakaoMapsReady) return
    // 지도 생성...
    setIsMapReady(true)
  }

  if (window.kakaoMapsReady) {
    initMap()
  } else {
    window.addEventListener('kakao-maps-loaded', initMap)
    // Fallback: 100ms 간격 폴링
  }
}, [])
```

**효과**:
- ✅ SDK 로드 완료 후에만 지도 생성
- ✅ 로딩 상태 정확하게 표시
- ✅ 이벤트 기반 + 폴링 fallback으로 안정성 확보

if (filters.status && filters.status !== 'all') {
