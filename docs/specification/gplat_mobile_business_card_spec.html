📋 프로젝트 개요

1. 서비스 소개

개인이 선택한 다양한 부업을 통합 관리하고 홍보할 수 있는 모바일 명함 서비스입니다.
사용자가 게시판 작성하듯 쉽게 부업 정보를 등록하면, 개인화된 한글 도메인 기반의 모바일 명함이 자동으로 생성 및 관리됩니다.

2. 타겟 고객

  • 연령대: 대학생부터 회사원, 자영업자까지 다양
  • 성별: 구분 없음
  • 관심사: 본업 홍보 및 부수입 창출을 위한 부업 홍보와 고객 확보

3. 핵심 목표

  • 쉽고 편한 부업 선택과 연동으로 추가적인 매출 증대
  • 기존 명함에서 확장된 명함 정보 전달과 본업의 홍보
  • 다양한 렌탈 서비스 홍보를 통한 추가 수입과 대상 고객 확보

✨ 서비스 차별화 포인트

1

간편한 등록 방식

  • 게시판 작성하듯 직관적인 인터페이스
  • 복잡한 커스터마이징 없이 필수 정보만 입력
2

한글 개인 도메인

  • https://김성근.한국 형태의 기억하기 쉬운 URL
  • 한글 도메인 전환 서비스 연동
3

모바일 최적화

  • 스마트폰에서 보기 편한 카드형 레이아웃
  • 터치 친화적 UI/UX 설계
4

콜백 기능

  • 통화/SMS 종료 후 자동으로 개인 URL 발송
  • 별도 조작 없이 원클릭 고객 연결
5

즉시 반영

  • 등록 즉시 개인 페이지에 실시간 반영
  • 실시간 수정 및 관리 가능

🏗️ 서비스 구조

1. 개인 모바일 페이지 구성

https://사용자명.한국 ├── 헤더 - 명함 섹션 │ └── 사각형 명함 카드 (이름, 직업, 회사명, 연락처) ├── 명함 상세 정보 │ ├── 상세 연락처 (이름, 직업, 회사명, 연락처, 주소, 이메일, 홈페이지) │ ├── QR코드 (개인별 URL) │ └── 추가 정보 ├── SNS 연동 버튼 │ └── 인스타그램, 페이스북, 유튜브, 블로그 └── 하단 네비게이션 메뉴 ├── 소개 (서비스 소개) ├── 이벤트 (협력 업체 이벤트) ├── 작성 (부업 카드 관리) ├── 공유 (다양한 공유 기능) └── 프로필 (개인 정보 관리)

2. 하단 네비게이션 메뉴 상세

1. 소개 메뉴

  • 서비스 개요 및 특징 소개
  • 단계별 사용법 가이드
  • 주요 기능 설명 및 예시
  • 실제 사용자 후기

2. 이벤트 메뉴

  • 협력 업체 이벤트 리스트
  • 이벤트 상세 정보 및 참여 방법
  • 부업 종류별 카테고리 필터
  • 관심 이벤트 즐겨찾기

3. 작성 메뉴 (핵심 기능)

  • 부업 카드 등록: 제목, 이미지, 설명, URL 입력
  • 부업 카드 관리: 수정, 삭제, 활성/비활성 설정
  • 통계 조회: 부업별 조회수, 클릭수 확인
  • 순서 변경: 드래그 앤 드롭으로 순서 조정

4. 공유 메뉴

  • 즉시 공유: 문자, 카카오톡, 이메일 등
  • SNS 공유: 인스타그램, 페이스북 등
  • 콜백 기능: 자동 URL 발송 시스템
  • QR코드 공유: QR코드 이미지 다운로드

5. 프로필 메뉴

  • 개인 정보 수정 (이름, 직업, 연락처, 소개글)
  • 명함 이미지 설정 (프로필 사진, 배경, 레이아웃)
  • 상세 정보 관리 (주소, 이메일, SNS 계정 연결)
  • 회원 추천 기능 (개인 추천 링크를 통한 회원 가입 기능)

📱 콜백 기능 상세

1. 동작 원리

2
시스템이 자동으로 해당 고객 번호를 감지
3
고객에게 https://김성근.한국 SMS 자동 발송

2. 기술적 특징

  • Zero-Touch: 사용자 별도 조작 불필요

  • 자동화: 통화/SMS 앱 연동을 통한 완전 자동화

3. 콜백 기능 상세
