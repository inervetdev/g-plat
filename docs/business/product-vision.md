# 지플랫 제품 비전 및 가치 제안

## 제품 정의

**지플랫(G-Plat)**은 개인 부업 포트폴리오를 통합 관리하고 홍보할 수 있는 모바일 명함 웹 빌더 서비스입니다. 사용자는 직관적인 인터페이스를 통해 모바일 명함을 생성하고, 다양한 부업 정보를 등록하여 한글 도메인 기반의 개인 브랜드를 구축할 수 있습니다.

## 제품 비전

> "모든 개인을 1인 기업가로, 명함 하나로 시작하는 부업 생태계"

## 핵심 가치 제안

### 1. 간편한 시작
- **3분 만에 완성되는 모바일 명함**
- 직관적인 UI/UX로 누구나 쉽게 생성
- 템플릿 기반 빠른 커스터마이징

### 2. 스마트한 관리
- **여러 부업을 하나의 플랫폼에서 통합 관리**
- 부업별 독립적인 카드 시스템
- 카테고리 기반 자동 분류 및 추천

### 3. 자동화된 마케팅
- **콜백 기능으로 자연스러운 고객 접점 확보**
- 통화 후 자동 SMS 발송
- 명함 링크 자동 전달

### 4. 데이터 기반 성장
- **실시간 통계로 비즈니스 인사이트 제공**
- 방문자 추적 및 분석
- QR 코드 스캔 통계
- 디바이스/브라우저별 분석

## 경쟁 우위

| 구분 | 리멤버 | 아정당 | 지플랫 |
|------|--------|--------|--------|
| 주요 기능 | 명함 관리 | 부업 중개 | 명함 + 부업 통합 |

| 실시간 대시보드 | 유료 | X | 기본 제공 |

## 차별화 포인트

### 1. 부업 생태계 통합
- 단순 명함이 아닌 **부업 포트폴리오 플랫폼**
- 카테고리별 부업 관리 (쇼핑, 교육, 서비스 등)
- 부업별 독립적인 CTA 및 추적

### 2. 한글 도메인 기반 브랜딩
- 개인화된 .한국 도메인 (예: 김대리.한국)
- 기억하기 쉬운 개인 브랜드
- 한국 시장 특화 전략

### 3. 자동화 마케팅
- 콜백 시스템으로 자연스러운 고객 관계 유지
- SMS 자동 발송으로 재접촉 보장
- 고객 데이터베이스 자동 구축

### 4. 풍부한 분석 기능
- 실시간 방문자 추적
- QR 코드별 스캔 통계
- 디바이스/브라우저/OS 분석
- 전환율 추적 (예정)

## 시장 포지셔닝

```
         고기능
            |
            |
      [기업용 CRM]
            |
            |
저가 -------- 지플랫 -------- 고가
      (Freemium)
            |
      [명함 앱들]
            |
         저기능
```

- **타겟**: 개인 부업자 및 1인 기업가
- **포지션**: 고기능 + Freemium 가격
- **전략**: 무료로 시작, 기능 확장으로 유료 전환

## 미래 비전

### 1단계 (현재): 명함 플랫폼
- 모바일 명함 생성 및 관리
- 부업 포트폴리오 통합
- 기본 통계 제공

### 2단계 (6개월): 마케팅 자동화
- 콜백 시스템 고도화
- SMS/이메일 마케팅 통합
- 고객 관계 관리 (CRM)

### 3단계 (12개월): 부업 생태계
- 부업 매칭 플랫폼
- 파트너사 중개 시스템
- 거래 수수료 수익 모델

### 4단계 (24개월): 1인 기업 플랫폼
- 팀 협업 기능
- 결제 통합
- 회계/세무 연동
- AI 기반 비즈니스 컨설팅
