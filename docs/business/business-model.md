# 지플랫 비즈니스 모델 및 수익 구조

## 비즈니스 목표

### 단기 목표 (3개월)
- MVP 출시 및 베타 테스트 완료
- 초기 사용자 1,000명 확보
- 핵심 기능 안정화
- 파트너사 5개 확보

### 중기 목표 (6개월)
- MAU 5,000명 달성
- 유료 전환율 10% 달성
- 파트너사 20개 확대
- 월 거래액 1억원 달성

### 장기 목표 (12개월)
- MAU 50,000명 달성
- 유료 전환율 15% 달성
- 파트너사 50개 이상
- 월 거래액 10억원 달성
- 시리즈 A 투자 유치

## 수익 모델

### 1. Freemium 구독 모델

#### 무료 플랜 (Free)
**가격**: ₩0/월

**기능**:
- 기본 명함 1개
- 부업 카드 3개
- 월 100회 콜백
- 기본 통계
- 5개 테마 선택

**제한**:
- 한글 도메인 미지원
- 고급 통계 미지원
- 브랜딩 제거 불가
- API 연동 불가

#### 프리미엄 플랜 (Premium)
**가격**: ₩9,900/월 (연 결제 시 ₩99,000/년, 16% 할인)

**기능**:
- 무제한 명함
- 무제한 부업 카드
- 무제한 콜백
- 고급 통계 (전환율, 유입 경로 등)
- 한글 도메인 (.한국)
- 브랜딩 제거
- 커스텀 테마 (향후)
- 우선 고객 지원

**타겟**: 개인 부업자, 프리랜서

#### 비즈니스 플랜 (Business)
**가격**: ₩29,900/월 (연 결제 시 ₩299,000/년, 16% 할인)

**기능**:
- Premium 플랜 모든 기능
- 팀 계정 (최대 5명)
- API 연동
- 전담 계정 관리자
- 맞춤형 리포트
- 화이트라벨 옵션
- 데이터 내보내기
- SLA 보장

**타겟**: 소규모 기업, 스타트업, 에이전시

### 2. 부업 중개 수수료

#### 렌탈 제품
- **수수료율**: 월 렌탈료의 5-15%
- **예시**: 정수기 (월 30,000원) → 수수료 1,500-4,500원/월
- **정산 주기**: 월별 정산

#### 일반 제품
- **수수료율**: 판매가의 3-10%
- **예시**: 건강식품 (100,000원) → 수수료 3,000-10,000원
- **정산 주기**: 판매 후 즉시

#### 보험 상품
- **수수료율**: 초회 보험료의 20-30%
- **예시**: 자동차 보험 (연 500,000원) → 수수료 100,000-150,000원
- **정산 주기**: 계약 성립 후 1개월 이내

### 3. 추가 수익원 (향후)

#### 프리미엄 템플릿 판매
- 개별 템플릿: ₩5,000-₩10,000
- 템플릿 번들: ₩20,000-₩50,000

#### 광고 수익
- 무료 플랜 사용자 명함 하단 광고 노출
- CPM (1,000회당 노출): ₩500-₩2,000

#### 데이터 분석 리포트
- 산업별 벤치마크 리포트: ₩50,000/건
- 맞춤형 시장 분석: ₩100,000-₩500,000/건

## 수익 예측

### 시나리오 1: 보수적 (12개월)
- 총 사용자: 10,000명
- 무료: 7,000명 (70%)
- 프리미엄: 2,500명 (25%)
- 비즈니스: 500명 (5%)

**월 구독 수익**:
- 프리미엄: 2,500 × ₩9,900 = ₩24,750,000
- 비즈니스: 500 × ₩29,900 = ₩14,950,000
- **합계**: ₩39,700,000/월

**연 수익**: ₩476,400,000

### 시나리오 2: 낙관적 (12개월)
- 총 사용자: 50,000명
- 무료: 32,500명 (65%)
- 프리미엄: 15,000명 (30%)
- 비즈니스: 2,500명 (5%)

**월 구독 수익**:
- 프리미엄: 15,000 × ₩9,900 = ₩148,500,000
- 비즈니스: 2,500 × ₩29,900 = ₩74,750,000
- **합계**: ₩223,250,000/월

**연 수익**: ₩2,679,000,000

## 성장 전략

### 1단계: 초기 사용자 확보 (0-3개월)
- 무료 플랜 중심 마케팅
- 추천 프로그램 (추천인/피추천인 각 1개월 무료)
- 인플루언서 협업
- 커뮤니티 마케팅 (LinkedIn, 브런치 등)

### 2단계: 유료 전환 (3-6개월)
- 프리미엄 기능 강화
- 한글 도메인 출시
- 콜백 자동화 완성
- 유료 전환 캠페인 (첫 달 50% 할인)

### 3단계: 수익 다각화 (6-12개월)
- 부업 중개 수수료 본격화
- 파트너사 확대
- 프리미엄 템플릿 출시
- 기업용 플랜 강화

## 투자 계획

### Seed 라운드 (목표: 3억원)
- 제품 개발 및 MVP 완성
- 초기 마케팅
- 팀 확장 (개발자 2명, 마케터 1명)
- 6개월 운영 비용

### Series A (목표: 20억원, 12개월 후)
- 시장 확대
- 한글 도메인 시스템 구축
- AI 기능 개발
- 팀 확장 (10명 추가)
